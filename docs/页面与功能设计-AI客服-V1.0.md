# AI 客服系统 - 页面与功能详细设计

**版本：** V1.0
**日期：** 2026-03-01

---

## 1. 用户端 (客户端)

### 1.1 微信小程序

**页面结构：**
```
首页
├── 聊天窗口 (核心)
│   ├── 消息列表 (气泡形式)
│   ├── 输入框 + 发送按钮
│   └── 常见问题快捷入口
├── 个人中心
│   ├── 历史对话
│   ├── 常见问题 (FAQ)
│   └── 联系人工客服
└── 帮助中心
```

**交互流程：**
1. 用户进入小程序 → 展示聊天窗口
2. 用户输入问题 → AI 即时回复
3. 无法回答时 → 显示"转人工"按钮
4. 人工回复 → 通知用户

---

### 1.2 飞书机器人

**功能：**
- 群聊中 @机器人 提问
- 私聊模式（一对一客服）
- 支持图片、文件上传
- 关键词触发自动回复

**配置：**
- 群聊：需把机器人拉入群
- 私聊：用户主动发起会话

---

### 1.3 Web 客服组件

**嵌入方式：**
```html
<script src="https://cdn.example.com/widget.js"></script>
<script>
  ChatWidget.init({
    appId: 'xxx',
    position: 'right-bottom', // left-bottom / right-bottom
    theme: 'light', // dark
    title: '在线客服'
  });
</script>
```

**界面：**
```
┌─────────────────────┐
│    在线客服    ✕    │  ← 头部
├─────────────────────┤
│                     │
│   [用户] 你好       │  ← 消息区域
│   [AI] 您好！      │
│                     │
├─────────────────────┤
│ 输入框...      发送  │  ← 输入区域
└─────────────────────┘
      (展开/收起)
```
